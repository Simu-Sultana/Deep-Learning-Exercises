## Overview

This repository focuses on implementing core deep learning concepts:

- Neural network architectures
- Model training and evaluation
- Loss function analysis
- Image-based datasets
- Debugging training pipelines

---

## Key Components

- `model.py` - Defines the neural network architecture
- `train.py` - Runs the training process
- `trainer.py` - Handles the training loop and evaluation
- `images/` - Contains the dataset images
- `losses.png` - Shows the training loss visualization

---

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/Simu-Sultana/Deep-Learning-Exercises.git
cd Deep-Learning-Exercises
```

### 2. Install dependencies

```bash
pip install torch torchvision numpy matplotlib
```

### 3. Run training

```bash
python exercise4_material/src_to_implement/train.py
```
